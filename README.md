## Description

Серверная часть тз для Автомат-Сервис

## Installation

```bash
$ npm install
```

## Docker

Все данные уже в папке data, нужно запустить docker

```bash
$ docker compose up
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
