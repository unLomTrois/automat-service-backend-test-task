## Description

Серверная часть тз для Автомат-Сервис

## Installation

```bash
$ npm install
```

## Docker

```bash
$ docker compose up
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Создание админа

http://localhost:3000/api - апи в сваггере, там есть метод Post для ресурса User, пароль хешируется
